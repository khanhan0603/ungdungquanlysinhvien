# Ứng dụng Quản Lý Sinh Viên (Android)

## Giới thiệu
Đây là **ứng dụng Android quản lý sinh viên**, phục vụ cho mục đích học tập và thực hành.  
Ứng dụng hỗ trợ các chức năng chính như:
- Đăng nhập hệ thống
- Xem danh sách sinh viên
- Xem / chỉnh sửa thông tin sinh viên
- Xem thông tin ứng dụng
- Tích hợp Google Maps (hiển thị vị trí)

Ứng dụng được xây dựng bằng **Android Studio**, sử dụng **Java** và kết nối dữ liệu từ Backend.

---

## Công nghệ sử dụng
- **Java (Android)**
- **Android Studio**
- **Gradle**
- **Google Maps SDK**
- **REST API**

---

## Yêu cầu hệ thống
- Android Studio (phiên bản mới)
- Android SDK
- JDK 8+
- Thiết bị Android hoặc Emulator
